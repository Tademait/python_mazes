# About the project

A Python library for generating mazes.

Example maze:
```
+---+---+---+---+---+
|                   |
+   +   +   +---+   +
|   |   |   |       |
+   +---+   +   +   +
|   |       |   |   |
+   +   +   +---+   +
|   |   |   |       |
+---+---+---+   +   +
|               |   |
+---+---+---+---+---+
```

# credits

Based on the book [Mazes for Programmers](http://www.mazesforprogrammers.com) from `Jamis Buck` (original examples are written in Ruby)
